# MS Excel Test

A test and practice repository for Microsoft Excel files, spreadsheet experiments, and analyst workflow validation.

## Overview

This repository is intended for testing Excel workbook structures, formulas, data cleaning steps, dashboards, and reporting formats before using them in a final portfolio or production-style repository.

## Key Skills Demonstrated

- Excel workbook testing
- Formula validation
- Pivot table and chart experiments
- Spreadsheet formatting and layout checks
- Data cleaning practice
- Business reporting workflow testing

## Repository Structure

This repository may include draft Excel files, sample datasets, test workbooks, screenshots, and notes. Since this is a test repository, files may be experimental and should be reviewed before being reused elsewhere.

## How to Use

1. Download or clone the repository.
2. Open the workbook files in Microsoft Excel.
3. Review formulas, sheet structure, and test outputs.
4. Use the files to validate Excel methods before applying them to final reports.
